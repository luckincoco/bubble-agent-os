## 正文

### 为什么 AI 记不住你？

用过 ChatGPT 的人都有这个体验：第一次对话觉得很惊艳，但用了一个月后发现，它对你的了解还停留在零。你的名字、你的工作、你的偏好——每次都要重新说一遍。

OpenAI 后来加了 Memory 功能，但本质上只是存了几条笔记。Google 的 NotebookLM 能导入文档做问答，但它不会主动学习。

**有没有一种 AI 助手，能像你的同事一样——聊得越多越懂你，不需要你反复解释？**

这就是我做泡泡 Agent OS 的初衷。

### 什么是"泡泡理论"

想象你的记忆是一个水族箱，里面漂浮着无数气泡：

- **新信息 = 新气泡**：浮在水面（工作记忆），非常活跃
- **时间流逝 = 气泡下沉**：没有被重复提及的信息逐渐淡化
- **相关气泡 = 自然聚合**：讨论同一主题的记忆聚在一起
- **聚合到一定程度 = 压缩成大泡泡**：从散碎事实变成一个有意义的理解

这不是隐喻——泡泡 Agent OS 就是这样工作的。

### 三个让我惊喜的功能

#### 1. 导入 Excel 后自动发现异常

我把公司的采购数据导入后，系统没等我问，就自动标记了几条异常记录：某个供应商的报价突然偏离了历史水平。

这是"惊讶检测器"在工作——它被动扫描每条新数据，和已有记忆对比，发现矛盾或异常就自动标记。

#### 2. 对话越多越懂你

连续使用两周后，压缩引擎把我 169 条散碎的 Excel 数据记录压缩成了一句话：

> "用户深度关注现金流健康和运营效率，监控频率暗示对财务风险的持续担忧。"

这不是我让它总结的——是系统在后台自动完成的抽象跃迁。从此以后，当我问模糊的问题（比如"最近有什么需要注意的？"），它会优先推送这类高阶洞察，而不是堆砌原始数据。

#### 3. 问"最近怎么样"不再是废话

传统 AI 收到"最近发生了什么"这种问题，要么胡说八道，要么说"我没有记忆"。

泡泡 Agent 会识别这是一个"时间型"查询，自动将时间衰减权重调到 50%，优先返回近期记忆。四种查询意图（精确/模糊/时间/聚合）有不同的权重配置，全自动切换。

### 部署体验：5 分钟搞定

```bash
git clone https://github.com/luckincoco/bubble-agent-os.git
cd bubble-agent-os
pnpm install
cp .env.example .env  # 填入 DeepSeek API Key（或用 Ollama 完全本地）
pnpm build:all
pnpm start --serve
# 打开 http://localhost:3000，默认账号 bobi / bubble123
```

整个项目只依赖 Node.js 和 SQLite，不需要 Docker、不需要 Postgres、不需要 Redis。一个端口搞定 API + WebSocket + 前端。

支持安装为手机 PWA——在手机浏览器里打开，点"添加到主屏幕"就行。

### 与 Notion AI / ChatGPT Memory 对比

| 能力 | ChatGPT Memory | Notion AI | 泡泡 Agent OS |
|------|---------------|-----------|--------------|
| 记忆方式 | 手动笔记 | 文档索引 | 自动提取 + 图谱关联 |
| 检索方式 | 不透明 | 向量搜索 | 四路融合（关键词+向量+图谱+时间） |
| 异常检测 | 无 | 无 | 自动（惊讶检测器） |
| 记忆压缩 | 无 | 无 | 分层抽象（原子→概念→画像） |
| 数据隐私 | 云端 | 云端 | 完全本地（SQLite + Ollama） |
| 自部署 | 不可 | 不可 | MIT 开源 |
