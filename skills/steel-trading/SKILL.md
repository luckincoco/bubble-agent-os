# 钢贸业务录入

当检测到用户消息中包含钢材品类关键词和数量/价格信息时，自动触发结构化录入流程。

## 检测规则

同时满足以下条件时触发：
1. 包含钢材品类词（螺纹、盘螺、高线、圆钢、线材、型材等）
2. 包含数量或价格信息（数字+吨/元/根/支）

## 处理流程

1. **检测** (detector.ts): 正则匹配品类+数量，零 LLM 成本
2. **解析** (parser.ts): 一次 LLM 调用，结构化提取字段
3. **存储** (store.ts): 创建 memory 泡泡，自动去重，生成确认回复

## 提取字段

| 字段 | 说明 | 示例 |
|------|------|------|
| bizType | 业务类型 | arrival/shipment/purchase/quote |
| steelType | 钢材品类 | 螺纹钢/盘螺/高线/圆钢 |
| spec | 规格 | Φ12/HRB400E/Q235B |
| quantity | 数量 | 50吨 |
| price | 单价 | 3850元/吨 |
| supplier | 供应商 | 示例公司 |
| project | 关联项目 | 示例项目B |
| date | 日期 | 2026-03-24 |
